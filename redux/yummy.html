  1. 根据连接判断一个页面是否显示，当显示的路径不是根路径的时候，侧边栏显示出来，否则当在主页的时候将侧边栏隐藏

    <Route render={({ location }) => {
           return location.pathname !== '/' ?

         }} />

    13 弹性页面，手机页面扩展时，扩展可用空间，利用flexbox 弹性盒子要设置在父级盒子上

    .home {
display:flex;
flex-direction:column;  以行为方向
height:100vh; 垂直方向100%
justify-content：space-between; 对齐的内容，两者之间的空间
justify-content: space-around; 对齐的内容，空间环绕
}

  }

    this.setState({
      isOpen: false
    })
  }
